---

## 📱 Overview

**Wellness** is an Android mobile application developed using **Kotlin** in **Android Studio** to help users maintain a balanced and healthy lifestyle.

The application combines habit tracking, mood journaling, hydration reminders, and wellness progress monitoring into a single platform, enabling users to build positive routines and improve their overall well-being.

Designed with a clean and user-friendly interface, Wellness encourages consistency, self-reflection, and healthy daily habits through simple yet effective tools.

---

## ✨ Features

### 🧩 Daily Habit Tracker

Build and maintain positive habits with ease.

* Add new habits
* Edit existing habits
* Delete unwanted habits
* Mark habits as completed
* Track daily progress and consistency
* Monitor personal wellness goals

Examples include:

* Drinking Water
* Exercise
* Meditation
* Reading
* Healthy Eating
* Walking

---

### 😊 Mood Journal

Track your emotional well-being every day.

* Record moods using emoji selections
* Add personal notes and reflections
* Automatically store date and time
* Maintain a digital mood diary
* Encourage emotional awareness and mindfulness

---

### 💧 Hydration Reminder System

Stay hydrated throughout the day.

* Set personalized water intake reminders
* Receive notification alerts at scheduled times
* Improve hydration habits
* Powered by Android AlarmManager

---

### 📊 Mood Trends & Insights

Visualize emotional patterns over time.

* View weekly mood history
* Analyze mood trends
* Identify emotional changes
* Support mental wellness monitoring

---

### 🏠 Home Screen Widget

Quickly monitor your daily progress.

* Displays habit completion percentage
* Updates progress dynamically
* Accessible directly from the device home screen
* No need to open the application

---

### 🎨 Modern User Interface

Built using Material Design principles.

* Clean layouts
* Intuitive navigation
* Responsive user experience
* Attractive and easy-to-use design

---

## 🗃️ Data Storage

The application uses **SharedPreferences** for local data persistence.

Stored data includes:

* User information
* Habit records
* Mood entries
* Reminder settings
* Application preferences

### Benefits

✅ Fast performance

✅ Offline functionality

✅ Lightweight storage

✅ No external database required

✅ Seamless user experience
